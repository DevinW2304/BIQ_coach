# HoopCoach AI Starter Package

## Downloadable package contents

- `python_app/` - the assignment-ready Python project
- `python_app/app/` - modular application code
- `python_app/assets/` - matching basketball-theme CSS and landing mockup
- `python_app/saved_workouts/` - output folder for exported workouts

## Main files

- `python_app/main.py` - program entry point
- `python_app/app/config.py` - environment loading and settings
- `python_app/app/prompts.py` - Mistral prompt builder
- `python_app/app/mistral_client.py` - official SDK integration
- `python_app/app/workout_generator.py` - workout generation and save helpers
- `python_app/app/ui.py` - terminal UI loop and validation
- `python_app/README.md` - setup and run guide
- `python_app/abstract_draft.md` - submission-ready abstract starter
- `python_app/example_inputs.md` - test scenarios

## Notes

This package is meant to be a polished starter you can submit and extend. The Python code is runnable after adding a valid Mistral API key in `.env`.
